📘 课程简介

本模块专为 Java 开发者 设计，帮助您系统深入地学习 FastAPI 框架。

🎯 学习目标

  • 掌握 FastAPI 高级路由与依赖注入模式
  • 理解异步编程在 Web 框架中的应用
  • 熟练使用 Pydantic 进行数据校验
  • 掌握认证授权的最佳实践
  • 学会编写可测试、可维护的 FastAPI 应用

📖 如何使用

  1. 从左侧菜单选择章节开始学习
  2. 完成理论学习后，尝试对应的实验室练习
  3. 使用 ✓ 标记已完成的内容
  4. 查看进度条追踪学习进展

💡 提示

每个章节都包含 Java vs Python 对比部分，帮助您快速建立心智模型。

🚀 快速开始

点击左侧菜单中的 "02_路由与依赖注入进阶" 开始第一课。

或者直接动手尝试 "lab01_router_splitting" 实验。
