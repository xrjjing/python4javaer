# 本地工具箱 - 使用指南

## 功能概述

本地工具箱是一个桌面应用，包含以下功能模块：

1. **凭证管理** - 存储和管理各类账号密码
2. **命令管理** - 保存常用命令，支持分页签管理
3. **节点转换** - 订阅链接解析与转换

## 运行方式

### 开发环境
```bash
cd desktop_tools
python main.py
```

### 打包后
- macOS: 双击 `本地工具箱.app`
- Windows: 双击 `本地工具箱.exe`

> 打包后的应用是独立的，目标电脑无需安装Python环境。

## 数据存储位置

| 环境 | 路径 |
|------|------|
| 开发环境 | `desktop_tools/电脑使用/` 和 `desktop_tools/转化节点/` |
| 打包后 | `~/.local_toolbox/电脑使用/` 和 `~/.local_toolbox/转化节点/` |

数据文件：
- `commands.json` - 命令数据
- `credentials.json` - 凭证数据
- `command_tabs.json` - 页签数据
- `nodes.md` - 节点数据

## 打包命令

```bash
cd desktop_tools
python build.py
```

打包产物位于 `dist/` 目录。

## 主题切换

点击右上角的主题图标可切换白天/夜间模式，默认为白天模式。
