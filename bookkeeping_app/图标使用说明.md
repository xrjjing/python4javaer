# 喵喵存金罐 - 图标使用说明

## 📁 文件结构

```
bookkeeping_app/
├── icon.png          # PNG 图标 (512x512) - 打包时使用
├── icon.ico          # Windows 图标 - 打包时使用
├── icon.icns         # macOS 图标 - 打包时使用
├── icons/
│   └── icon.svg      # SVG 源文件 - 可编辑的矢量图
└── build.py          # 打包脚本
```

## 🎨 图标设计

- **主题**: 可爱的猫咪 + 金币
- **配色**: 粉色背景 (#FFB6C1 → #FF69B4)
- **风格**: 现代、可爱、记账概念
- **尺寸**: 512x512 像素

## 🔧 修改图标

### 1. 编辑 SVG 源文件

```bash
# 使用任意 SVG 编辑器打开
open icons/icon.svg
```

推荐编辑器：
- **Figma** (在线)
- **Inkscape** (免费)
- **Adobe Illustrator**
- **Sketch** (macOS)

### 2. 转换为多种格式

修改完 SVG 后，运行转换脚本：

```bash
# 在项目根目录运行
python convert_icons.py
```

脚本会自动：
1. 将 SVG 转换为 PNG (512x512)
2. 生成 ICO 文件 (Windows 多尺寸图标)
3. 生成 ICNS 文件 (macOS 多尺寸图标)
4. 将所有文件输出到应用根目录

### 3. 重新打包应用

```bash
cd bookkeeping_app
python build.py
```

打包后的应用会使用新图标。

## 📋 注意事项

1. **不要删除根目录的图标文件**
   - `icon.png`, `icon.ico`, `icon.icns` 是打包脚本直接引用的
   - 删除后打包会失败

2. **只编辑 SVG 源文件**
   - 修改 `icons/icon.svg`
   - 不要直接修改 PNG/ICO/ICNS

3. **保持图标尺寸**
   - 建议保持 512x512 的画布尺寸
   - 图标内容居中，留适当边距

4. **转换脚本依赖**
   - 需要安装 ImageMagick: `brew install imagemagick`
   - 需要 Python 包: `wand`, `pillow` (脚本会自动安装)

## 🚀 快速开始

```bash
# 1. 修改图标
open icons/icon.svg

# 2. 转换格式
python ../convert_icons.py

# 3. 重新打包
python build.py

# 4. 查看效果
open dist/喵喵存金罐/喵喵存金罐.app
```

## 🎯 图标设计建议

- **简洁明了**: 图标在小尺寸下也要清晰可辨
- **品牌一致**: 保持猫咪主题和记账概念
- **颜色鲜明**: 使用高对比度，便于识别
- **适配深色模式**: 确保在深色背景下也好看
- **可爱风格**: 符合"喵喵存金罐"的品牌调性

## 🎨 主题系统

喵喵存金罐支持多主题切换，图标设计应考虑：

- **light** (浅色): 明亮清新
- **cute** (可爱): 粉色系，默认主题
- **office** (办公): 专业简洁
- **dark** (深色): 护眼舒适
- **neon** (霓虹): 炫彩夜间
- **cyberpunk** (赛博朋克): 科技感

## 📞 问题排查

### 转换脚本报错

```bash
# 检查 ImageMagick 是否安装
convert --version

# 重新安装
brew install imagemagick

# 检查 Python 依赖
pip list | grep -E "wand|pillow"
```

### 打包后图标未更新

1. 清理旧的打包文件: `rm -rf dist build *.spec`
2. 确认根目录图标文件已更新
3. 重新运行 `python build.py`

### macOS 图标缓存问题

```bash
# 清除图标缓存
sudo rm -rf /Library/Caches/com.apple.iconservices.store
killall Dock
```
