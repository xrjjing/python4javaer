# 03_条件判断与分支

## 学习目标

- 熟练使用 `if / elif / else` 结构
- 理解缩进对分支结构的影响
- 理解「真值」概念（哪些值会被当做 False）

---

## 知识点清单

- 基本结构：

  ```python
  if 条件:
      ...
  elif 其他条件:
      ...
  else:
      ...
  ```

- 真值规则：
  - 以下会被当做 False：`0`、`0.0`、空字符串 `""`、空列表 `[]`、空字典 `{}`、`None`
  - 其他大多数对象都被当做 True

---

## 示例 demo：成绩等级判断

```python
# demo_if.py

score_text = input("请输入成绩（0-100）：")
score = float(score_text)

if score >= 90:
    level = "A"
elif score >= 80:
    level = "B"
elif score >= 70:
    level = "C"
elif score >= 60:
    level = "D"
else:
    level = "E"

print("成绩等级：", level)
```
