# 13_更多运算符与表达式细节

> 这一章主要补齐一些容易被忽略但在实际开发中很常用的语法点：`in`、`is`、运算符优先级、`pass` 等。

## 学习目标

- 理解成员运算符 `in / not in`
- 理解身份运算符 `is / is not`
- 知道常见运算符的优先级，避免歧义
- 了解 `pass` 语句的用途

---

## 知识点清单

### 1. 成员运算符 `in / not in`

```python
nums = [1, 2, 3]
print(2 in nums)        # True
print(4 not in nums)    # True

text = "python"
print("py" in text)     # True
```

### 2. 身份运算符 `is / is not`

- 判断的是「是否是同一个对象」，而不是值是否相等。

```python
a = [1, 2, 3]
b = a
c = [1, 2, 3]

print(a == c)   # True，值相等
print(a is c)   # False，不是同一个对象
print(a is b)   # True
```

### 3. 运算符优先级

- 大致记住几个常见规则：
  - 括号最高：`( )`
  - 乘除优先于加减
  - 比较运算符在算术运算之后
  - 逻辑运算：`not` > `and` > `or`
- 为了可读性，复杂表达式建议用括号明确优先级。

### 4. `pass` 语句

- 占位语句，什么也不做，只是为了保证语法结构完整。

```python
def todo():
    pass  # 以后再实现
```

## 示例 demo：综合使用

```python
# demo_more_ops.py

def is_valid_name(name: str) -> bool:
    if not name:
        return False
    if " " in name:
        return False
    return True


def main():
    names = ["alice", "", "bob chen"]
    for n in names:
        if not is_valid_name(n):
            print("非法名字：", n)
        else:
            print("合法名字：", n)


if __name__ == "__main__":
    main()
```
