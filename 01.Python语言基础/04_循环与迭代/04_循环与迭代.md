# 04_循环与迭代

## 学习目标

- 掌握 `while` 和 `for` 循环语法
- 理解 `range()` 的用法
- 熟悉 `break` 与 `continue` 的行为

---

## 知识点清单

- `while 条件:`：只要条件为真就反复执行
- `for 变量 in 可迭代对象:`：遍历序列或其他可迭代对象
- `range(start, stop, step)`：生成整数序列
- `break`：提前跳出当前循环
- `continue`：跳过本次循环剩余语句，进入下一次

---

## 示例 demo：累加与简单菜单

```python
# demo_loop.py

# 1 到 100 的累加
total = 0
for i in range(1, 101):
    total += i
print("1 到 100 的和：", total)

# 简单菜单
while True:
    print("1. 查询余额")
    print("2. 存款")
    print("3. 退出")
    choice = input("请选择操作：")

    if choice == "1":
        print("余额功能暂未实现")
    elif choice == "2":
        print("存款功能暂未实现")
    elif choice == "3":
        print("已退出")
        break
    else:
        print("无效选择，请重试")
```
