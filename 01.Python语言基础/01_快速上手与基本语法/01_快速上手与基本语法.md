# 01_快速上手与基本语法

## 学习目标

- 知道如何运行 Python 脚本（命令行 / PyCharm）
- 理解缩进在 Python 中的作用（代替花括号）
- 掌握注释、`print` 输出、`input` 输入的基本用法

---

## 知识点清单

- Python 执行方式：`python script.py`
- 缩进：使用 4 个空格表示代码块
- 注释：行注释 `# ...`
- 基本输入输出：`print()`、`input()`

---

## 示例 demo：简单交互脚本

```python
# demo_hello.py

# 询问用户姓名并问好
name = input("请输入你的名字：")
print("你好，", name)

# 简单算式
print("1 + 2 的结果是：", 1 + 2)
```

在终端中运行（确保已经在项目根目录并激活虚拟环境）：

```bash
python 01.Python语言基础/01_快速上手与基本语法/demo_hello.py
```

> 提示：你也可以直接在 PyCharm 中右键 `Run` 来运行脚本。

## 练习题（建议新建 `practice_basic_io.py` 完成）

1. 编写程序，依次从键盘输入你的姓名、年龄、所在城市，并按如下格式输出：  
   `我是张三，今年 25 岁，来自北京。`
2. 编写程序，输入两个数字，输出它们的和、差、积、商。  
3. 编写程序，输入一个边长，输出正方形的周长和面积。  
4. 思考题：如果用户输入的是文本 `"10"`，它在 Python 中是什么类型？如何把它转换为整数再参与计算？
