# 09_面向对象入门

## 学习目标

- 理解在 Python 中定义类和创建对象的方式
- 掌握构造方法 `__init__` 与实例属性
- 能编写简单的方法，并在对象上调用
- 初步理解继承与方法重写（对比 Java）

---

## 知识点清单

- 类定义：

  ```python
  class User:
      def __init__(self, name, age):
          self.name = name
          self.age = age

      def greet(self):
          print(f"你好，我是{self.name}")
  ```

- 创建对象：`user = User("张三", 25)`
- 访问属性：`user.name`
- 调用方法：`user.greet()`
- 简单继承：

  ```python
  class Student(User):
      def __init__(self, name, age, school):
          super().__init__(name, age)
          self.school = school
  ```

---

## 示例 demo：用户与学生

```python
# demo_oop.py

class User:
    def __init__(self, name: str, age: int):
        self.name = name
        self.age = age

    def greet(self) -> None:
        print(f"你好，我是{self.name}，今年{self.age}岁。")


class Student(User):
    def __init__(self, name: str, age: int, school: str):
        super().__init__(name, age)
        self.school = school

    def greet(self) -> None:
        # 覆盖父类方法
        print(f"大家好，我是{self.name}，来自{self.school}，今年{self.age}岁。")


if __name__ == "__main__":
    u = User("张三", 30)
    u.greet()

    s = Student("李四", 20, "清华大学")
    s.greet()
```
