# 02_变量_数据类型_运算符

## 学习目标

- 理解 Python 变量的本质（绑定名称到对象）
- 熟悉常见内置数据类型：整数、浮点数、布尔、字符串
- 掌握算术、比较、逻辑运算符的用法
- 会做基本的类型转换（`int` / `float` / `str`）

---

## 知识点清单

- 变量命名规则与规范（下划线命名法，区分大小写）
- 数值类型：`int`、`float`
- 布尔类型：`bool`，只有 `True` 和 `False`
- 字符串：`str`，单引号或双引号都可以
- 运算符：
  - 算术：`+ - * / // % **`
  - 比较：`== != > >= < <=`
  - 逻辑：`and`、`or`、`not`
- 类型转换函数：`int()`、`float()`、`str()`

---

## 示例 demo：变量与运算

```python
# demo_types_ops.py

age = 28          # 整数
height = 1.75     # 浮点数（米）
name = "Alice"    # 字符串
is_student = False  # 布尔值

print("类型：", type(age), type(height), type(name), type(is_student))

# 算术运算
year = 2025
birth_year = year - age
print("出生年份：", birth_year)

# 比较与逻辑运算
print("是否成年：", age >= 18)
print("既是学生又满 18 岁：", is_student and age >= 18)

# 类型转换
text_number = "10"
num = int(text_number)
print("字符串转整数后加 5：", num + 5)
```
