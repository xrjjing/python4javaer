# 06_函数与参数

## 学习目标

- 能够定义和调用函数
- 理解返回值与无返回值的区别
- 熟练使用默认参数、关键字参数、可变参数
- 初步理解作用域（局部变量与全局变量）

---

## 知识点清单

- 基本函数定义：

  ```python
  def add(a, b):
      return a + b
  ```

- 返回值：`return`；没有 `return` 或 `return` 后无值时，返回 `None`
- 参数类型：
  - 位置参数
  - 默认参数
  - 关键字参数
  - `*args`、`**kwargs`
- 作用域：局部变量、`global` 关键词（实际开发中应尽量少用）

---

## 示例 demo：日志函数与计算函数

```python
# demo_functions.py

def log(message, prefix="[INFO]"):
    print(prefix, message)


def add(*numbers):
    """求多个数字的和"""
    total = 0
    for n in numbers:
        total += n
    return total


def say_hello(name, greeting="你好"):
    print(f"{greeting}, {name}")


if __name__ == "__main__":
    log("程序开始")
    result = add(1, 2, 3, 4)
    log(f"结果是 {result}")
    say_hello("张三")
    say_hello("李四", greeting="Hello")
```
