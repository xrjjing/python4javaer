# 01.Python 语言基础（针对 Java 开发者）

> 目标：在不引入框架和项目实战的前提下，系统掌握 Python 的核心语法与常用内置类型，为后续 Web / 脚本 / 数据方向打下坚实基础。

学习顺序建议按章节从上到下依次完成，每一章都有：

- 学习目标
- 知识点清单
- 一个简短 demo
- 一组练习题（自行在本章目录下新建 `.py` 文件完成）

---

## 目录结构总览

- `01_快速上手与基本语法`  
  安装运行、Hello World、缩进、注释、输入输出、简单脚本。

- `02_变量_数据类型_运算符`  
  变量命名、动态类型、数字 / 布尔 / 字符串、运算符、类型转换。

- `03_条件判断与分支`  
  `if / elif / else`、布尔表达式、常见坑（空字符串、空列表的真值）。

- `04_循环与迭代`  
  `while`、`for`、`range`、`break / continue`、嵌套循环。

- `05_序列与集合类型`  
  `list`、`tuple`、`dict`、`set`，增删改查，切片，基础推导式。

- `06_函数与参数`  
  `def`、返回值、位置参数 / 关键字参数 / 默认参数、`*args` / `**kwargs`、作用域。

- `07_模块与包基础`  
  `import`、`from ... import ...`、`__name__ == "__main__"`、简单脚本组织。

- `08_异常与文件基础`  
  `try / except / else / finally`、`raise`、`open / read / write`、`with`。

- `09_面向对象入门`  
  `class`、`__init__`、实例方法、属性、简单继承，对比 Java 类。

- `10_迭代器与生成器`  
  可迭代对象、迭代器、`iter` / `next`、生成器函数与生成器表达式。

- `11_lambda表达式与装饰器`  
  匿名函数、常见使用场景、简单函数装饰器（计时、日志等）。

- `12_命名空间_作用域_类型注解`  
  LEGB 规则、`global` / `nonlocal`、基础类型注解、`typing` 模块。

- `13_更多运算符与表达式细节`  
  成员运算符 `in` / `not in`，身份运算符 `is`，运算符优先级，`pass` 占位语句。

- `14_开发技巧_写法优化`  
  f-string、推导式、enumerate / zip、any / all、字典高级用法、pathlib、logging、解包运算符等 Pythonic 写法建议。

> 使用建议：每完成一章，就在本文件或者章节 README 里勾选对应自检项，并在该章节目录下建 `practice_xx.py` 等文件写练习答案。
