# 02_推导式_列表字典集合

## 技巧说明

- 列表 / 字典 / 集合推导式用于从一个可迭代对象快速构造新的集合；
- 写法紧凑，但仍然保持可读性；
- 适合「过滤 + 映射」这种典型场景。

---

## 示例 demo（见 `demo_comprehensions.py`）

示例涵盖：

- 列表推导式：平方、过滤偶数；
- 字典推导式：构造 key → value 映射；
- 集合推导式：快速去重。

---

## 练习建议

1. 将你之前写过的「用 for 循环构造新列表」改写成列表推导式；  
2. 在日志分析项目中，用字典推导式构建「日志级别 → 次数」的排序结果；  
3. 在监控项目中，用集合推导式得到「出现过的指标名」集合。
