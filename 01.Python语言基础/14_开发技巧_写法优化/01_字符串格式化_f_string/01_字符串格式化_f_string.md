# 01_字符串格式化_f_string

## 技巧说明

- 使用 f-string（格式化字符串字面量）替代字符串拼接；
- 支持在 `{}` 中直接写变量和表达式；
- 支持格式控制（小数位数、对齐等）。

Python 3.6+ 开始支持 f-string，几乎所有新代码都推荐使用。

---

## 示例 demo（见本目录 `demo_f_string.py`）

主要示例：

- 基本用法：`f"你好，{name}"`；
- 表达式：`f"2 + 3 = {2 + 3}"`；
- 小数格式：`f"价格：{price:.2f}"`。

---

## 练习建议

1. 把你在前面章节写过的「个人信息输出」练习改成 f-string 写法；  
2. 在日志分析 / 监控项目中，将复杂的字符串拼接重写为 f-string；  
3. 尝试使用宽度与对齐格式，例如：`f"{name:>10}"`、`f"{score:06d}"`。
