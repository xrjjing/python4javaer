# 07_模块与包基础

## 学习目标

- 理解「模块」和「包」的概念
- 学会使用 `import` 和 `from ... import ...`
- 理解 `__name__ == "__main__"` 的用途
- 能把简单程序拆分成多个文件组织

---

## 知识点清单

- 一个 `.py` 文件就是一个模块
- 一个包含 `__init__.py` 的目录可以视为一个包（现代 Python 不强制，但保留习惯有利于结构清晰）
- 导入方式：

  ```python
  import math
  from datetime import datetime
  ```

- `__name__`：当前模块的名字  
  - 直接运行脚本时：`__name__ == "__main__"`  
  - 被其他模块导入时：`__name__ == 模块名`

---

## 示例 demo：拆分工具模块

```python
# utils.py

def greet(name: str) -> None:
    print(f"你好，{name}")


def add(a: int, b: int) -> int:
    return a + b
```

```python
# demo_import.py

from utils import greet, add


def main():
    greet("张三")
    result = add(3, 5)
    print("3 + 5 =", result)


if __name__ == "__main__":
    main()
```

> 建议：把以上两个文件放在 `01.Python语言基础/07_模块与包基础/` 目录下，然后在该目录中运行 `python demo_import.py`。
