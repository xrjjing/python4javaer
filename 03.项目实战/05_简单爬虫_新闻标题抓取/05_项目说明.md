# 05_简单爬虫（基础教学版）

> 定位：最小化爬虫示例，聚焦 requests + BeautifulSoup 基础解析，用于入门。

## 主要特点
- 仅包含同步抓取与基本解析，逻辑简洁，便于阅读。
- 默认需要联网抓取目标 URL；无离线 sample。
- 解析策略：遍历所有 `<a>` 标签，过滤长度 < 6 的文本。

## 快速运行
```bash
python crawler_news_titles.py https://example.com
```

## 测试
```bash
pytest 03.项目实战/05_简单爬虫_新闻标题抓取/test_crawler_news_titles.py -q
```

## 与进阶版的区别
- 进阶版：`03.项目实战/09_新闻爬虫_Requests_BS4`，提供离线 sample、异步 httpx、限速与 robots 检查。
- 基础版：保留最简单的同步写法，适合作为“第一支爬虫”示例。
