### 1. 确认 Python 版本

```bash

```

### 2. 创建虚拟环境

```bash

python -m venv .venv
```

## 💡 常见问题

### Q1: 代码运行报错找不到模块？

**A:** 确保已激活虚拟环境并安装依赖

```bash
source .venv/bin/activate
pip install -r requirements.txt  # 如果有
```

### Q2: 如何在PyCharm中运行？

**A:** 配置Python解释器

1. Settings → Project → Python Interpreter
2. 选择 `.venv/bin/python`
3. 右键代码文件 → Run

### Q3: 学习顺序可以调整吗？

**A:** 可以！建议：

- **必学**：第1-6章（基础语法）
- **推荐**：第7-10章（进阶特性）
- **选学**：第11-13章（根据需要）
- **实战**：第14章技巧 + 项目实战

### Q4: 代码示例可以修改吗？

**A:** 当然！建议：

1. 先运行原始代码理解功能
2. 修改参数观察变化
3. 添加新功能扩展
4. 用Git管理你的修改
